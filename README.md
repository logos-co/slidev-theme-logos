# slidev-theme-logos

Logos brand theme for [Slidev](https://sli.dev).

## Usage

In your slides frontmatter:

```yaml
---
theme: ./slidev-theme-logos
---
```

Or if installed as a package:

```yaml
---
theme: logos
---
```

## Layouts

- `default` — Standard slide with Logos.co branding and page number
- `cover` — Title/cover slide, vertically centered, no page number
- `center` — Centered content (demos, statements)
- `two-cols` — Two-column layout with header (use `::left::` and `::right::` slots)
- `end` — Closing slide

## Design

- Dark background (#0A0A0A)
- Inter font family
- Logos.co branding top-left
- Page numbers bottom-left
- Minimal, monochrome palette
